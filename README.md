# 🐳 Introdução ao Docker

![Docker](https://www.docker.com/sites/default/files/d8/2019-07/Moby-logo.png)

O **Docker** é uma plataforma para **contêineres**, permitindo empacotar aplicações com todas as dependências em **containers isolados**, portáveis e leves.

---

## 🚀 Vantagens do Docker

- Isolamento de aplicativos  
- Portabilidade entre ambientes  
- Rápida inicialização de serviços  
- Reduz conflitos de dependências  
- Integração com CI/CD e orquestradores (Kubernetes)

---

## 💡 Comandos básicos

| Comando              | Função                                |
|---------------------|---------------------------------------|
| `docker build`      | Cria uma imagem a partir de um Dockerfile |
| `docker run`        | Executa um container                   |
| `docker ps`         | Lista containers ativos                |
| `docker stop`       | Para um container                      |
| `docker rm`         | Remove um container                     |
| `docker images`     | Lista imagens disponíveis              |
| `docker rmi`        | Remove imagem                           |
